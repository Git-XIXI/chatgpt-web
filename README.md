# chatgpt-web
#### 介绍
基于 Spring Boot + RabbitMQ + React + ChatGLM 的生成式 AI 助手，采用 DDD 架构，包含流式对话、敏感词过滤、分布式限流、公众号鉴权登录、微信支付等功能。
<br>该工程为客户端代码。
